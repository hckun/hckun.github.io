hexo+github搭建一个博客

搭建一个博客
技术性
Publish Date:   2020-11-22
Update Date:   2020-11-23
Word Count:   866
Read Count:  

如果出现提示找不到啥啥命令自己npm下载去，不会就去百度找下载命令是什么~~~

安装Node.js

百度直接搜索node.js，选择长期支持版，安装选项一路默认

打开命令行，输入node -v和npm -v，如果出现版本号，那么就安装成功了。

换国内源

命令行输入 

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装Git

百度搜索git，一路默认

安装完成后在命令行中输入git --version验证是否安装成功。

命令行里输入

npm install --save hexo-deployer-git
npm install --save hexo-deployer-git

Github部分

新建一个仓库，仓库的项目名一定是 

你的github的用户名.github.io

除了用户名什么都不用写直接创建就ok了。

开始搭建本地博客

新建一个文件夹用来存放自己的博客文件

命令行cd 文件夹地址进入这个文件夹

输入npm i hexo-cli -g安装Hexo，输入hexo -v验证是否安装成功。

输入hexo init初始化文件夹(输入这个命令后文件夹会多出一些东西。注意：必须是空文件夹才可以初始化，要从来就要把文件夹里的东西删除，查看隐藏文件，有一个git文件会隐藏起来)。

如果这个时候hexo init卡在的话（卡住了文件夹里也会有一些已经下载好的文件，删了，注意那个隐藏的git，漏删了就不是空了），ctrl+z取消，在命令行中输入

git config --global --unset http.proxy
git config --global --unset https.proxy

然后再重试hexo init，不行就多来几遍。

好了之后，接着输入npm install安装必备的组件。

这样本地的网站配置就OK了，命令行中输入hexo g生成静态网页，然后输入hexo s打开本地服务器，然后浏览器打开http://localhost:4000/，就可以看到我们的博客的样子了。

按ctrl+c关闭本地服务器。

连接Github与本地

命令行cd 进入你的博客文件夹，输入下面命令：

双引号不用删除哈，下面都是这种写法

git config --global user.name "你的github用户名"
git config --global user.email "你绑定的的邮箱"

然后生成密钥SSH key（用来连接本地和github的协议）：

ssh-keygen -t rsa -C "你的邮箱"

打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字可以不写也可以随便写，下面的框先放着，进行下一步。

打开你的博客文件夹，右键git bush here输入

cat ~/.ssh/id_rsa.pub

将输出的内容复制到githubssh的框中，点击确定保存。

输入ssh -T git@github.com，如果提示 HI，啥啥啥！ you are successfully 巴拉巴拉巴拉巴拉 ，那就是成功了。

配置博客根目录下的_config.yml文件

修改最后一行的配置：

deploy:
  type: git
  repository: https://github.com/你的用户名/你的用户名.github.io.git
  branch: master

修改# URL文件的 中的url 和 root

url: https://你的用户名.github.io/ 
root: /你的用户名.github.io

部署到远端

命令行cd 进入博客文件夹

hexo clean //清理缓存
hexo g //生成静态网页
hexo d //上传到github

这样就大功告成！网页输入https://你的用户名.github.io/就可以进入你的博客了！
